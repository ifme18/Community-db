# flask_backend

Simple Flask project layout.

Structure:

flask_backend/
├── app/
│   ├── __init__.py
│   ├── models.py
│   ├── routes.py
│   └── config.py
├── venv/  # placeholder
├── run.py
└── requirements.txt

To run:
1. Create a virtualenv and install requirements: python -m venv venv && venv/bin/pip install -r requirements.txt
2. Run: venv/bin/python run.py
